# Voice Dictation

<span class="badge-teal">No Claude Code required</span>

I basically never type anymore. Voice dictation — talking instead of typing, in any app — is the single biggest productivity change I've made alongside AI chatbots.

**My current tool: [Wispr Flow](https://wisprflow.ai).** Works across Mac, Windows, and iPhone in every app — email, VS Code, Claude Code, web forms, everything. It learns my vocabulary over time.

!!! note "The landscape is changing fast"
    New dictation tools appear constantly. Claude is adding voice input. ChatGPT already has it in-app. What matters is the capability — system-wide voice-to-text that works reliably — not the specific product.

---

## Why Dictation Changes Everything

I type at maybe 60 words per minute on a good day. I speak at 150+. That's a 2-3x speed increase on any text-heavy task — and it eliminates the friction of starting. It's much easier to *talk through* an idea than to *write* it from scratch.

**The real payoff isn't speed — it's starting.** The hardest part of writing an email, a proposal section, or a meeting summary is getting the first words down. Dictation makes that effortless.

## How I Use It

**Email.** I dictate almost every email. Quick replies go directly into Gmail. Longer messages I dictate into Claude and let it format the response.

**AI prompts.** Dictate a rough request into Claude Code → use `/prompt` to reformat → get polished output. The whole sequence takes 30 seconds for a task that would take 5 minutes typing.

**Writing first drafts.** For any first draft — proposal sections, paper introductions, blog posts — I dictate a rough version and then edit. Dramatically faster than staring at a blank page.

## Alternatives

| Tool | Platform | Price | Notes |
|------|----------|-------|-------|
| **Apple Dictation** | Mac/iOS | Free | Built-in. Decent for short bursts. Doesn't learn vocabulary. |
| [**Superwhisper**](https://superwhisper.com/) | Mac/iOS | $10/mo | Local AI processing (privacy-focused). Supports offline use. |
| [**Aqua Voice**](https://www.aquavoice.com/) | Mac/Windows | $8/mo | Extremely fast (sub-50ms). System-wide. |
| [**Voibe**](https://www.getvoibe.com/) | Mac | Free-$10/mo | 100% offline, on-device processing. |
| **Google Voice Typing** | Any (via Google Docs) | Free | Not system-wide. |
| [**Whisper**](https://github.com/openai/whisper) (OpenAI) | Any (via apps) | Free/varies | Open-source. Requires technical setup. |

Any dictation tool is better than no dictation tool. Wispr Flow's advantage is that it works everywhere without switching apps.

## Getting Started

1. **Sign up** at [wisprflow.ai](https://wisprflow.ai/)
2. **Set your hotkey** — I use a dedicated key, but any modifier combo works
3. **Train it** — correct mistakes in the first few days and it learns
4. **Use it for everything** — the more you use it, the better it gets

The learning curve is about a week. After that, typing feels painfully slow.
